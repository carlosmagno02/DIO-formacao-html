História dos computadores

O primeiro computador criado foi o Eniac entre 1943 e 1946, tinha um custo muito alto e era utilizado apenas por empresas

O que são Clients?

Os clients são onde vemos o front end das aplicações, toda a parte visual

O que são Servers?

Os servers são onde acontece todo o processacemnto de informações da aplicação a "parte de tras da aplicação", tambem conhecido como back-end

Como é uma estrutura básica de um HTML?

A estrutura basica de um html é composta pelas tags essenciais para criar um site que são elas html,head e body>

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas as "ol" são listas que indicam alguma ordem podendo por exemplo ser uma ordem numerica, ja as "ul" são listas que não seguem nescessariamente uma ordem, conhecidas como listas não ordenadas
